Install Hermes Agent
One command — handles everything automatically
+100 XP
▼

Hermes installs with a single command. The installer automatically handles Python, Node.js, and all dependencies. No admin rights required on Windows.

Windows (PowerShell):

iex (irm https://hermes-agent.nousresearch.com/install.ps1)

Mac / Linux / WSL2:

curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
source ~/.bashrc
What the installer does
  • Checks for Python 3.11, Node.js, Git — installs if missing
  • Downloads and installs the Hermes Agent runtime
  • Creates config directory: %LOCALAPPDATA%\hermes\ (Windows) or ~/.hermes/ (Mac/Linux)
  • Adds hermes to your PATH automatically

Verify the install worked:

hermes --version

You should see a version number like hermes 1.x.x. If so, you're ready!

Run the Setup Wizard
Connect Hermes to an AI model in 2 minutes
+100 XP
🔒

Hermes needs an AI model to think with. The fastest way is to connect to Nous Portal (free OAuth — covers 300+ models, web search, image generation). Alternatively use OpenRouter with your own API key.

Option A — Nous Portal (Recommended, Free):

hermes setup --portal

This opens a browser window. Sign in with Google/GitHub. Done — no API key needed.

Option B — OpenRouter (Use any model):

hermes setup

When prompted, choose OpenRouter as the provider and paste your API key from openrouter.ai.

Tip — Which option to choose?
  • Nous Portal — best for beginners, no billing setup, works immediately
  • OpenRouter — best if you want full model choice (Claude, GPT-4, Gemini, etc.)
  • You can always switch later with hermes setup

Check Everything Works
Run the doctor and check your model connection
+100 XP
🔒

Use Hermes's built-in diagnostic tool to confirm everything is wired up correctly.

hermes doctor

This checks your install, config, model connection, and tool availability. You should see all green checkmarks.

hermes model

This shows which AI model is currently selected and lets you switch. Confirm you see a model listed.

Common issues
  • hermes: command not found — close and reopen your terminal, then try again
  • API key error — run hermes setup again and re-enter your key
  • Python not found — run the installer again; it will install Python automatically

Explore the Built-in Tools

Hermes comes with 60+ built-in tools out of the box. You can see and manage them with one command.

hermes tools

This lists all available tools. Browse the categories:

CategoryWhat it can do
WebSearch the web, fetch pages, extract data from URLs
FilesRead, write, edit, rename, search files on your computer
CodeRun Python/JS/Bash, analyse code, find bugs
MediaImage generation, text-to-speech, audio processing
MemoryStore facts, recall past sessions, build knowledge base
MessagingSend via Telegram, Discord, Slack, WhatsApp, Email

Enable or disable individual tools:

hermes tools enable web-search
hermes tools disable text-to-speech

Install a Plugin from the Community
Extend Hermes with community-built capabilities
+100 XP
🔒

Hermes has a plugin system — community members publish extensions that add new tools, integrations, and workflows. Install one to see how it works.

Install the model-sherpa safety plugin (adds model switching and safety middleware):

hermes plugins install nordicnode/model-sherpa

List all installed plugins:

hermes plugins list

Update Hermes to the latest version:

hermes update
Plugin ideas for PM / Business users
  • Search the Hermes plugin community for project management plugins
  • Look for email integration plugins to send reports via Gmail/Outlook
  • Find spreadsheet plugins for automated Excel/Sheets workflows
  • Or build your own and share it with the community!
