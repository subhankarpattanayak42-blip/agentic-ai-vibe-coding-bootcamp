Install Open Claw on Windows

Step-by-step guide — no technical background needed

Before You Start — You Need

  • A Windows 10 or Windows 11 computer
  • Internet connection
  • A Claude.ai account (Pro, Max, Team, or Enterprise plan)
  • About 10 minutes
1

Open PowerShell

PowerShell is a built-in Windows tool — no download needed. Here's how to open it:

Press the Windows key on your keyboard → type PowerShell → click Windows PowerShell

Tip: You do NOT need to right-click "Run as Administrator" — a normal PowerShell window works fine.

A blue (or dark) window will open with a blinking cursor. That's PowerShell — you're in the right place.

2

Run the Installer

Copy the command below and paste it into PowerShell, then press Enter:

PowerShell
irm https://claude.ai/install.ps1 | iex
How to paste in PowerShell: Right-click anywhere in the window — it pastes automatically. Or use Ctrl + Shift + V.

The installer will download and set up Open Claw automatically. Wait for it to finish — it usually takes 1–2 minutes.

3

Verify the Installation

Once the installer finishes, type this command and press Enter:

PowerShell
claude --version

You should see a version number printed (e.g. claude 2.x.x). That means Open Claw is installed correctly.

If you see "command not found": Close PowerShell, reopen it, and try again. The installation needs a fresh terminal to take effect.
4

Launch Open Claw

Navigate to the folder where your project is (or just stay in the home directory for now), then type:

PowerShell
claude

Open Claw will open in your browser and ask you to log in with your Claude account.

5

Log In with Your Claude Account

A browser window will open automatically. Log in with the same email and password you use on claude.ai.

Important: You need a Pro, Max, Team, or Enterprise plan. The free Claude.ai plan does not include Open Claw access.

Once logged in, go back to PowerShell — Open Claw is now ready to use.

6

Create Your AGENTS.md File (Open Claw Config)

Open Claw looks for an AGENTS.md file in your project folder to understand how to behave. Create one like this:

PowerShell — create a project folder and AGENTS.md
mkdir my-project
cd my-project
new-item AGENTS.md

Open the file in Notepad and add your instructions — for example, what language to use, how to name files, etc.

Tip: AGENTS.md is the Open Claw equivalent of a rulebook. Think of it as telling the AI: "Here's how I want you to work in this project."
7

Optional — Install Git for Windows (Recommended)

Git gives Open Claw a full Bash terminal, which makes it much more powerful. It's free and takes 2 minutes to install.

Download from: https://git-scm.com/downloads/win → run the installer → click Next all the way through.

Why bother? Without Git, Open Claw can only use PowerShell commands. With Git, it can run the full range of shell commands — much more capable.
🎉

You're Ready!

Open Claw is installed on your Windows machine. Type claude in PowerShell from any project folder to start building.

Quick Reference

Start Open Claw claude
Check version claude --version
Config file AGENTS.md
Run health check claude doctor
Uninstall Remove-Item "$env:USERPROFILE\.local\bin\claude.exe"
