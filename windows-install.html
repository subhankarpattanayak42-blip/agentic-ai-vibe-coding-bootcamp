Install Open Claw on Windows

with OpenRouter

Use any AI model — GPT-4o, Gemini, Llama, Mistral and more

Open Source · No single-vendor lock-in

What is Open Claw?

Open Claw is an open-source, self-hosted AI gateway that runs on your own machine. It connects to any AI model provider and lets you chat, code, and build agents through a browser dashboard or any messaging app. Full docs at docs.openclaw.ai

What is OpenRouter?

OpenRouter is a single API that gives you access to 200+ AI models — GPT-4o, Gemini, Llama, Mistral, DeepSeek and more — all with one key and one bill.

GPT-4o Gemini 2.5 Pro Llama 3.3 70B Mistral Large DeepSeek R1 Kimi K2 + 200 more

Before You Start — You Need

  • Windows 10 or Windows 11
  • Internet connection
  • Node.js 22.19+ or Node 24 installed (step 1 below)
  • An OpenRouter account — free at openrouter.ai
  • About 15 minutes
1

Install Node.js

Open Claw requires Node.js. Go to nodejs.org → download Node 24 (LTS) → run the installer → click Next all the way through.

Once done, verify it installed correctly. Open PowerShell (Windows key → type PowerShell → Enter) and run:

PowerShell node --version

You should see something like v24.x.x. If you do, Node.js is ready.

How to paste in PowerShell: Right-click anywhere in the window. Or use Ctrl + Shift + V.
2

Install Open Claw

Run the official Open Claw installer for Windows in PowerShell:

PowerShell iwr -useb https://openclaw.ai/install.ps1 | iex

Wait for it to complete — usually takes 1–2 minutes. Then verify:

PowerShell openclaw --version

You should see a version number. If so, Open Claw is installed.

If you see "command not found": Close PowerShell, reopen it, and try again — the install needs a fresh terminal to take effect.
3

Get Your OpenRouter API Key

Go to openrouter.ai → Sign up (free) → Click your profile icon → Keys → Create Key → copy it.

Your key looks like: sk-or-v1-xxxxxxxxxxxxxxxx

Add credits: Go to openrouter.ai → Settings → Credits → Add $5 to start. Most models cost fractions of a cent per message.
4

Run the Onboarding Wizard

Open Claw has a built-in setup wizard. Run this command and it will guide you through connecting to OpenRouter:

PowerShell openclaw onboard --auth-choice openrouter-api-key

When prompted, paste your OpenRouter API key (sk-or-v1-...) and press Enter. The wizard will configure everything automatically in about 2 minutes.

5

Install the Daemon and Open the Dashboard

This starts Open Claw as a background service and opens the browser dashboard:

PowerShell — install daemon openclaw onboard --install-daemon
PowerShell — open dashboard openclaw dashboard

Your browser will open at http://127.0.0.1:18789 — this is your Open Claw control panel. You can chat with your AI model directly from here.

6

Choose Your AI Model

By default Open Claw uses openrouter/auto which picks the best model automatically. To set a specific model:

PowerShell — set model # GPT-4o (OpenAI)
openclaw models set openrouter/openai/gpt-4o

# Gemini 2.5 Pro (Google)
openclaw models set openrouter/google/gemini-2.5-pro

# Llama 3.3 70B (Free tier)
openclaw models set openrouter/meta-llama/llama-3.3-70b-instruct:free

# DeepSeek R1 (low cost, high capability)
openclaw models set openrouter/deepseek/deepseek-r1
Browse all available models at openrouter.ai/models — filter by free, by cost, or by provider.
7

Check Gateway Status

Verify everything is running correctly:

PowerShell openclaw gateway status

You should see the gateway is listening on port 18789. If it shows running — you're all set!

🎉

You're Ready!

Open Claw is running on your Windows machine, powered by OpenRouter. Open the dashboard anytime with openclaw dashboard and start building with any AI model.
