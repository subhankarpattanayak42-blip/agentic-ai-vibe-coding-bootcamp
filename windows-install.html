Challenge 1 of 7
Open Your Terminal
+100 XP
🔓

Every quest starts with opening your command terminal. Press the Windows key on your keyboard, type PowerShell, and click Windows PowerShell.

A dark window opens with a blinking cursor. That's your command terminal — the gateway to everything.

To paste in PowerShell: Right-click anywhere in the window. Or use Ctrl + Shift + V.

✅ You're done when: You see a blinking cursor in a dark window.

⚙️
Challenge 2 of 7
Install Node.js
+100 XP
🔒

Open Claw needs Node.js to run. Go to nodejs.org, download Node 24 (LTS), run the installer, click Next all the way through.

Then verify it worked by running this in PowerShell:

PowerShell node --version

✅ You're done when: You see v24.x.x printed in the terminal.

Seeing an error? Close PowerShell, reopen it, and try again — Node needs a fresh terminal after install.
🦞
Challenge 3 of 7
Install Open Claw
+100 XP
🔒

Time to install Open Claw itself. Copy and paste this one command into PowerShell:

PowerShell — official installer iwr -useb https://openclaw.ai/install.ps1 | iex

Wait 1–2 minutes for it to finish. Then verify:

PowerShell openclaw --version

✅ You're done when: You see an Open Claw version number.

Command not found? Close PowerShell, reopen it, and try openclaw --version again.
🔑
Challenge 4 of 7
Get Your OpenRouter Key
+100 XP
🔒

OpenRouter gives you access to 200+ AI models with one key. Go to openrouter.ai → Sign up (free) → Profile icon → Keys → Create Key → Copy it.

Your key looks like: sk-or-v1-xxxxxxxxxxxxxxxx

Add $5 credits: openrouter.ai → Settings → Credits. Most models cost fractions of a cent per message. Free models are also available!
Models you'll be able to use:
GPT-4o Gemini 2.5 Pro Llama 3.3 70B (Free) DeepSeek R1 Mistral Large + 200 more

✅ You're done when: You have your sk-or-v1-... key copied.

🧙
Challenge 5 of 7
Run the Setup Wizard
+100 XP
🔒

Open Claw's onboarding wizard connects it to OpenRouter. Run this command — it will ask for your API key:

PowerShell openclaw onboard --auth-choice openrouter-api-key

When prompted, paste your sk-or-v1-... key and press Enter. The wizard configures everything in ~2 minutes.

Then install the background service:

PowerShell openclaw onboard --install-daemon

✅ You're done when: The wizard completes without errors.

🖥️
Challenge 6 of 7
Open the Dashboard
+100 XP
🔒

Launch your Open Claw control panel in the browser:

PowerShell openclaw dashboard

Your browser opens at http://127.0.0.1:18789 — this is your personal AI command centre. Everything you need to manage models, sessions, and agents is here.

Bookmark it: Save http://127.0.0.1:18789 in your browser for quick access.

✅ You're done when: The Open Claw dashboard loads in your browser.

🤖
Challenge 7 of 7 — FINAL BOSS
Send Your First AI Message
+100 XP
🔒

The final challenge. In the Open Claw dashboard, find the chat box and type your first message to the AI:

Type this in the dashboard chat Hello! What AI model are you? What can you help me build?

If you want to switch to a specific model first, run this in PowerShell:

PowerShell — optional: pick a model # Free option — Llama 3.3 70B
openclaw models set openrouter/meta-llama/llama-3.3-70b-instruct:free

# Or GPT-4o if you have credits
openclaw models set openrouter/openai/gpt-4o

✅ You're done when: The AI replies in the dashboard. You are now a Claw Champion!
