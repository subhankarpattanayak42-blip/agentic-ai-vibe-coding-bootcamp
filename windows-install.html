Open Claw on Windows

with OpenRouter

Use any AI model — GPT-4o, Gemini, Llama, Mistral — not just Claude

No Anthropic subscription needed

What is OpenRouter?

OpenRouter is a single API that gives you access to 200+ AI models from different providers — all with one API key and one bill. Instead of paying separately for Claude, GPT-4, Gemini etc., you top up OpenRouter credits and use any model.

GPT-4o Gemini 2.5 Pro Claude Sonnet Llama 3.3 70B Mistral Large DeepSeek R1 Qwen 2.5 + 200 more

Before You Start — You Need

  • A Windows 10 or Windows 11 computer
  • Internet connection
  • An OpenRouter account (free to sign up at openrouter.ai)
  • About 15 minutes
1

Open PowerShell

Press the Windows key → type PowerShell → click Windows PowerShell.

A blue/dark window opens with a blinking cursor. You do not need to run as Administrator.

2

Install Open Claw

Copy and paste this command into PowerShell, then press Enter:

PowerShell irm https://claude.ai/install.ps1 | iex
How to paste in PowerShell: Right-click anywhere in the window — it pastes automatically. Or use Ctrl + Shift + V.

Wait 1–2 minutes for the installation to finish.

3

Get Your OpenRouter API Key

Go to openrouter.ai → Sign up (it's free) → Click your profile → Keys → Create Key.

Copy the key — it looks like: sk-or-v1-xxxxxxxxxxxxxxxx

Add credits: Go to openrouter.ai → Settings → Credits → Add $5–$10 to start. Most models cost fractions of a cent per request.
4

Set OpenRouter as Your Provider

This tells Open Claw to use OpenRouter instead of Anthropic. Run both commands in PowerShell (replace the key with yours):

PowerShell — set environment variables $env:ANTHROPIC_BASE_URL = "https://openrouter.ai/api/v1"
$env:ANTHROPIC_API_KEY = "sk-or-v1-your-key-here"
Note: These variables last only for this PowerShell session. See Step 6 to make them permanent.
5

Choose Your Model

Tell Open Claw which model to use. This is the fun part — pick any model from OpenRouter:

PowerShell — popular free/cheap options # GPT-4o (OpenAI)
$env:CLAUDE_MODEL = "openai/gpt-4o"

# Gemini 2.5 Pro (Google)
$env:CLAUDE_MODEL = "google/gemini-2.5-pro"

# Llama 3.3 70B (free on OpenRouter)
$env:CLAUDE_MODEL = "meta-llama/llama-3.3-70b-instruct:free"

# DeepSeek R1 (very capable, low cost)
$env:CLAUDE_MODEL = "deepseek/deepseek-r1"
Browse all models at openrouter.ai/models — filter by free, by price, or by provider.
6

Make Settings Permanent (Recommended)

So you don't have to re-enter the keys every time, save them permanently. Run this in PowerShell:

PowerShell — save permanently [System.Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://openrouter.ai/api/v1", "User")
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-or-v1-your-key-here", "User")

Close and reopen PowerShell after running this. The settings will now load automatically every time.

7

Launch Open Claw

Navigate to any project folder (or stay where you are) and type:

PowerShell claude

Open Claw will start and connect to OpenRouter. You'll see a prompt where you can start typing your first instruction.

First thing to try: Type What model are you using? — it should respond with the OpenRouter model you selected.
8

Create Your AGENTS.md Config File

Open Claw looks for an AGENTS.md file in your project folder. This is where you give it standing instructions.

PowerShell — create a project and AGENTS.md mkdir my-project
cd my-project
New-Item AGENTS.md -ItemType File

Open AGENTS.md in Notepad and write your rules — for example:

AGENTS.md — example content # My Project Rules
- Use Python 3.10+
- Write clean, commented code
- Always create a requirements.txt
- Keep responses concise
🎉

You're Ready!

Open Claw is running on your Windows machine, powered by OpenRouter. Switch models anytime by changing ANTHROPIC_MODEL — no reinstall needed.
