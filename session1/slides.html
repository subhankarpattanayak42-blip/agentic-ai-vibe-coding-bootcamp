Building Blocks05 / 08

The 5 Building Blocks

🤖 Agents
Autonomous systems that perceive environment, make decisions, and take actions. The LLM is the brain; the agent is the whole system.
🎛️ Orchestrators
Control systems managing agent workflows — decide which tools to use, in what order. Like a project manager directing specialists.
🔧 Tools
Functions/APIs the agent can invoke: file ops, database queries, web search, code execution. Each tool has a schema the agent uses.
🧠 Memory
Short-term: current context window. Long-term: persistent storage. Challenge: memory costs tokens.
🔄 Reasoning Loops
Think → Plan → Execute → Evaluate → Decide next step. Enables self-correction. Trade-off: more loops = better reasoning but higher cost & latency.
